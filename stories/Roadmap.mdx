# Ant Design Component Gap Roadmap

This document lists the components missing from the current React UI library compared to the latest Ant Design version, grouped by priority to guide development.

---

# 1. Must-have Components

These components unlock **80–90% of real-world application UIs**, especially dashboards, admin panels, and complex forms.

## Data Display
- Table
- Card
- Badge
- Divider
- Progress
- Empty

## Navigation
- Menu
- Pagination
- Anchor

## Feedback
- Drawer
- Popconfirm
- Spin (if Spinner is not equivalent)

## Data Entry
- AutoComplete
- Upload
- Slider
- Rate

## Layout
- Space
- Layout

---

# 2. Should-have Components

These components are useful in **feature-rich enterprise applications** but are not strictly required for most projects.

## Data Display
- Collapse
- Descriptions
- List
- Statistic
- Timeline
- Image

## Data Entry
- Cascader
- Transfer
- Mentions
- ColorPicker

## Feedback
- Notification
- Message
- Tour

## Navigation
- FloatButton

---

# 3. Nice-to-have Components

These components are **specialized or rarely required**, but help achieve full parity with Ant Design.

## Data Display
- Calendar
- Carousel
- QRCode
- Tree
- Segmented

## Layout
- Splitter
- Masonry

## Feedback
- Watermark

## Other
- Affix
- ConfigProvider
- App
- Util
