# Pagination spec

## Purpose
Theme-aware pagination component modeled after Ant Design pagination behavior and props, implemented with local UI primitives.

## API highlights
- Supports controlled/uncontrolled `current` and `pageSize`.
- Supports `showSizeChanger`, `showQuickJumper`, `showTotal`, `simple`, `hideOnSinglePage`.
- Supports `itemRender`, `pageSizeOptions`, and locale strings similar to Ant Design.

## Composition
- Uses `Button` for page items and nav controls.
- Uses `Select` for page-size changer.
- Uses `Input` for quick jumper and simple mode editable page.
