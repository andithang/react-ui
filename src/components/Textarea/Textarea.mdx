# Textarea spec

## Purpose
Render multiline text input with label, hint, and error feedback.

## Props
- label?: `string`
- hint?: `string`
- error?: `string`
- Inherits native textarea props via `TextareaHTMLAttributes<HTMLTextAreaElement>`

## Behavior
- Renders label, textarea, and helper/error text.
- Applies `.ui-control--error` on error state.
- Error text takes precedence over hint text.
- Textarea is vertically resizable with minimum height.

## Accessibility
- Uses native `<textarea>`.
- Label wraps the control for accessible naming.
- Keyboard behavior is native.

## Styling
- Uses `.ui-textarea`, `.ui-textarea__error`, and shared `.ui-control`.
- Reuses shared label/helptext typography classes.

## Deliverables
- React component
- SCSS style file
- Storybook coverage in Inputs stories
