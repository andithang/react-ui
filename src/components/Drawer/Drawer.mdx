# Drawer spec

## Purpose
Slide-out panel container based on Ant Design Drawer API shape.

## Props
- Placement and sizing: `placement`, `size`, `width`, `height`.
- Visibility lifecycle: `open`, `afterOpenChange`, `destroyOnClose`.
- Close behavior: `onClose`, `keyboard`, `closable`, `closeIcon`, `mask`, `maskClosable`.
- Composition slots: `title`, `extra`, `footer`, `children`, `classNames`, `styles`.

## Accessibility
- Uses `role="dialog"` and Escape support when `keyboard` is enabled.
