# Table spec

## Purpose
Ant Design-inspired data table with generic typing and column definitions for rendering structured records.

## Supported Ant-style API surface
- `columns`, `dataSource`, `rowKey`
- `size`, `bordered`, `loading`, `showHeader`, `tableLayout`, `scroll`
- `pagination` (`current`, `pageSize`, `total`, size changer callbacks)
- `title`, `summary`, `footer`, `locale.emptyText`
- `rowClassName`, `onRow`, `onHeaderRow`, `onChange`

## Column capabilities
- `title`, `key`, `dataIndex`
- `render`, `align`, width props, `ellipsis`
- `sorter`, `sortOrder`, `defaultSortOrder`, `sortDirections`
- `filters`, `filteredValue`, `defaultFilteredValue`, `onFilter`
- `onCell`, `onHeaderCell`

## Behavior
- Uses semantic native table structure (`table`, `thead`, `tbody`, `th`, `td`).
- Supports client-side sorting, filtering, and pagination patterns.
- Exposes Ant-like callback shape through `onChange`.

## Styling
- Root class: `.ui-table`
- Internal classes: `.ui-table__native`, `.ui-table__head-cell`, `.ui-table__cell`, `.ui-table__pagination`
- Uses theme tokens and primary-color interaction states.
