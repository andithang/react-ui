# ThemeSwitch spec

## Purpose
Provide a convenience switch that toggles light/dark theme from context.

## Props
- label?: `string`

## Behavior
- Reads `theme` and `toggleTheme` from `useThemeContext`.
- Renders `Switch` with checked state bound to `theme === 'dark'`.
- Default label is `Dark mode`.

## Accessibility
- Delegates accessibility semantics to underlying `Switch`.
- Consumers can customize label text for clarity.

## Styling
- Inherits styling from `Switch`; no local SCSS file.

## Deliverables
- React component
- Storybook usage in Foundation/Theming
