# DatePicker spec

## Purpose
Ant Design DatePicker wrapper with full AntD picker behavior and configurable props.

## API parity
- `DatePicker` now uses Ant Design's DatePicker implementation directly.
- Supports AntD date value model (`dayjs`) and all AntD configurable props.
- Includes static pickers from AntD:
  - `DatePicker.RangePicker`
  - `DatePicker.WeekPicker`
  - `DatePicker.MonthPicker`
  - `DatePicker.YearPicker`
  - `DatePicker.QuarterPicker`

## Exported types
- `DatePickerProps`
- `RangePickerProps`
- `MonthPickerProps`
- `WeekPickerProps`
- `YearPickerProps`
- `QuarterPickerProps`

## Styling
- Uses Ant Design structural behavior.
- Applies library token overrides for colors, borders, focus ring, selected states, and dropdown surfaces.
- Primary interactive highlight is `var(--ui-primary)` (`#e03` in light theme).
