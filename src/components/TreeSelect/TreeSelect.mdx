# TreeSelect spec

## Purpose
Provide an Ant Design-inspired `TreeSelect` primitive for common application flows.

## Props
- Includes common props aligned with Ant Design patterns (status/size/disabled/className and behavior-specific options).
- Refer to `TreeSelect.tsx` for the complete TypeScript prop interface used by this library build.

## Behavior
- Uses token-based styling and supports light/dark themes.
- Supports controlled/uncontrolled patterns where applicable.

## Accessibility
- Uses semantic HTML and keyboard-friendly interactions for interactive controls.

## Styling
- Main class follows the `ui-` prefix convention via the component stylesheet.
