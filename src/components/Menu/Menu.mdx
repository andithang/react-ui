# Menu spec

## Purpose
Ant Design-like navigation `Menu` with support for item groups, nested menus, selection, and inline/horizontal layout.

## Props
- `items` supports item, submenu, group, and divider nodes.
- Controlled and uncontrolled state: `selectedKeys`/`defaultSelectedKeys`, `openKeys`/`defaultOpenKeys`.
- Behavior options: `mode`, `theme`, `selectable`, `multiple`, `inlineCollapsed`, `triggerSubMenuAction`.
- Event API: `onClick`, `onSelect`, `onDeselect`, `onOpenChange`.

## Styling
- Uses `ui-menu` prefixed classes.
- Hover, active, and focus states use semantic primary tokens.
