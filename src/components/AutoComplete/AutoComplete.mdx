# AutoComplete spec

## Purpose
Ant Design-style autocomplete input built from the shared `Input` component.

## Props
- Accepts all `Input` props plus Ant-like autocomplete API:
  `options`, `open/defaultOpen`, `filterOption`, `onSearch`, `onSelect`, `onOpenChange`, `notFoundContent`, `popupClassName`, `dropdownMatchSelectWidth`.
- Supports controlled and uncontrolled value/open states.

## Behavior
- Filters options by value text by default.
- Keyboard support for arrow navigation, Enter select, Escape close.
