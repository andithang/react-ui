# Badge

`Badge` displays counts, status dots, or ribbons attached to content.

## Props

- `count`, `overflowCount`, `showZero`, `dot`, `size`
- `status`, `color`, `text`
- `offset`, `title`
- `classNames`, `styles`

## Ribbon

Use `Badge.Ribbon` with `text`, `placement` (`start`/`end`) and optional `color`.
