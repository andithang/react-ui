# Spinner spec

## Purpose
Display a visual loading indicator.

## Props
- size?: `number`
- Inherits native span props via `HTMLAttributes<HTMLSpanElement>`

## Behavior
- Renders a single `<span>` with spinner styles.
- Maps `size` to `--spinner-size` CSS variable.
- Uses CSS animation for rotation.

## Accessibility
- Decorative by default; add `aria-label` or status wrapper when semantic loading feedback is needed.

## Styling
- Uses `.ui-spinner`.
- Color and sizing are token-based.

## Deliverables
- React component
- SCSS style file
- Storybook usage in Feedback story
