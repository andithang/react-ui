# Switch spec

## Purpose
Render a binary on/off toggle using a checkbox input.

## Props
- label?: `string`
- Inherits native checkbox input props via `InputHTMLAttributes<HTMLInputElement>`

## Behavior
- Renders hidden native checkbox with custom track/thumb visuals.
- Checked state moves thumb and changes track color.
- Supports controlled and uncontrolled usage.

## Accessibility
- Uses native `<input type="checkbox">`.
- Label click toggles state.
- Keyboard accessible by default.

## Styling
- Uses `.ui-switch`, `.ui-switch__input`, `.ui-switch__track`, `.ui-switch__thumb`.
- Label uses `.ui-label`.

## Deliverables
- React component
- SCSS style file
- Storybook usage in Inputs and Theming stories
