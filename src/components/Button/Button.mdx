# Button spec

## Purpose
Ant Design-compatible action trigger with configurable type/color/variant, loading, icon placement, shape, and link rendering.

## Visual behavior
- `type`: `default | primary | dashed | link | text`
- `color`: `default | primary | danger | blue | purple | cyan | green | magenta | pink | red | orange | yellow | volcano | geekblue | lime | gold`
- `variant`: `outlined | dashed | solid | filled | text | link`
- `shape`: `default | circle | round | square`
- `size`: `small | middle | medium | large`
- Supports `ghost`, `danger`, `block`, `icon`, `iconPlacement`

## Props
- `type?: ButtonType`
- `color?: ButtonColorType`
- `variant?: ButtonVariantType`
- `icon?: ReactNode`
- `iconPlacement?: 'start' | 'end'`
- `iconPosition?: 'start' | 'end'` (deprecated compatibility alias)
- `shape?: ButtonShape`
- `size?: SizeType`
- `loading?: boolean | { delay?: number; icon?: ReactNode }`
- `disabled?: boolean`
- `ghost?: boolean`
- `danger?: boolean`
- `block?: boolean`
- `href?: string` (renders as anchor)
- `htmlType?: 'submit' | 'button' | 'reset'`
- `autoInsertSpace?: boolean`
- `classNames?: { root?: string; icon?: string; content?: string } | ((info) => ...)`
- `styles?: { root?: CSSProperties; icon?: CSSProperties; content?: CSSProperties } | ((info) => ...)`
- `prefixCls?: string`
- `rootClassName?: string`
- `children?: ReactNode`
- Inherits native `button` + `anchor` HTML attributes

## Behavior
- Uses native `<button>` by default and `<a>` when `href` is provided.
- Loading state supports immediate or delayed activation.
- Clicks are blocked while `loading` or `disabled`.
- Supports Ant-style icon rendering and icon end placement.
- Supports Ant-style grouped buttons via `Button.Group`.

## Accessibility
- Preserves native button semantics when rendered as `<button>`.
- Anchor mode uses `aria-disabled` and removes tab focus when disabled.
- Icon-only usage should include an accessible name (for example `aria-label`).

## Styling
- Root class: `.ui-button`
- Key state classes:
  - `.ui-button-color-*`
  - `.ui-button-variant-*`
  - `.ui-button-loading`
  - `.ui-button-background-ghost`
  - `.ui-button-icon-only`
  - `.ui-button-icon-end`
  - `.ui-button-block`
- Group class: `.ui-button-group`
