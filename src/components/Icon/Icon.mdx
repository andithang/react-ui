# Icon spec

## Purpose
Render reusable SVG icons from an internal icon path map.

## Available icons
- search
- close
- check
- chevronDown
- info
- sun
- moon
- alert

## Props
- name: `IconName` (required)
- size?: `number`
- Inherits native SVG props via `SVGProps<SVGSVGElement>`

## Behavior
- Renders `<svg>` with `fill="currentColor"`.
- Uses `size` for both width and height.
- Default size is `20`.

## Accessibility
- Sets `aria-hidden="true"` by default.
- For semantic icons, consumers should pass proper ARIA props.

## Styling
- Uses `.ui-icon` class and consumer-provided `className`.
- Color inherits from surrounding text unless overridden.

## Deliverables
- React component
- Internal icon path map
- Story usage across component demos
