# Tabs spec

## Purpose
Render tabbed navigation and panel content from an item list.

## Data model
- `TabItem`
- id: `string`
- label: `ReactNode`
- content: `ReactNode`

## Props
- items: `TabItem[]`
- defaultTabId?: `string`
- className?: `string`

## Behavior
- Returns `null` when `items` is empty.
- Uses `defaultTabId` or first item id as initial active tab.
- Clicking a tab sets it active and renders its `content`.
- Active tab is resolved with fallback to first item.

## Accessibility
- Uses `role="tablist"`, `role="tab"`, and `role="tabpanel"`.
- Sets `aria-selected` on active tab.
- Keyboard arrow-key behavior is not implemented yet.

## Styling
- Uses `.ui-tabs`, `.ui-tabs__list`, `.ui-tabs__tab`, `.ui-tabs__panel`.
- Active tab uses `.is-active`.

## Deliverables
- React component
- SCSS style file
- Storybook tabs story
