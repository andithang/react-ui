# Anchor spec

## Purpose
Page navigation helper that tracks and highlights the active heading link.

## Props
- `items` supports nested anchor links (Ant Design-like structure).
- Scroll/offset control: `bounds`, `offsetTop`, `targetOffset`, `getContainer`, `getCurrentAnchor`.
- Layout options: `affix`, `direction`, `replace`.
- Events: `onChange`, `onClick`.

## Styling
- Uses `ui-anchor` classes and semantic primary states for hover/active/focus.
