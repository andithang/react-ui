# Tooltip spec

## Purpose
Show contextual text/content on hover or focus-within.

## Props
- content: `ReactNode`
- children: `ReactNode`
- position?: `'top' | 'right' | 'bottom' | 'left'`
- className?: `string`

## Behavior
- Wraps trigger content and tooltip bubble.
- Bubble visibility is controlled by CSS on hover/focus-within.
- Default position is `top`.

## Accessibility
- Bubble uses `role="tooltip"`.
- Trigger semantics depend on provided `children`.
- Consumers should ensure focusable trigger content when keyboard access is required.

## Styling
- Uses `.ui-tooltip` and `.ui-tooltip__bubble`.
- Position modifiers: `.ui-tooltip__bubble--top|right|bottom|left`.

## Deliverables
- React component
- SCSS style file
- Storybook usage in Feedback story
